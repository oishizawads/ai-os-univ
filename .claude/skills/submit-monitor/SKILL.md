# Submit Monitor

## Before Submit
- format 確認
- inference 整合確認
- experiment id 記録
- result.md 更新
- submission filename 命名確認

## After Submit
- public score 記録
- local CV との差分確認
- 採用/保留/破棄を判定
- session notes 更新

## Output
- Submission status
- Risks
- Record checklist
