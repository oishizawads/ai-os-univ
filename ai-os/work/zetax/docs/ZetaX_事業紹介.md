## 3つのサービス

### 1. PQC（耐量子暗号）

**背景・危機感**
- 2035年：現行暗号システムのデッドライン（量子コンピュータによる解読リスク）
- G7・BIS（国際決済銀行）がすでに対応済み
- 国内企業の対応は大幅に遅れている

**サービス内容**

| メニュー | 内容 |
|---------|------|
| PQC研修 | 耐量子暗号の基礎から実装まで |
| 脆弱性診断 | 現行システムの量子脅威リスク評価 |
| 徹底導入支援 | PQCへの移行・実装サポート |

---

### 2. AI DX 支援（オーダーメイド型）

**特徴**: 汎用ツール導入ではなく、クライアント課題に合わせたフルカスタム開発

**導入事例**

| クライアント | 課題 | 解決策 |
|-------------|------|--------|
| 大手製薬会社 | 販売戦略の属人化 | 販売戦略AIの構築 |
| サブスクEC | レコメンドの手動運用 | レコメンドエンジン自動化 |
| 中堅企業 | 問い合わせ対応コスト | カスタマーサポートbot開発 |

**営業フロー**

```
ヒアリング → 提案 → PoC・開発 → 導入
```

---

### 3. Reservoir Computing（RC）の社会実装

**連携先**: 東京工業大学研究室

**RC の特性（従来MLとの差別化）**

| 特性 | 内容 |
|------|------|
| 少データ対応 | 大量の学習データが不要 |
| 低消費電力 | エッジデバイス向け |
| 時系列処理に強い | センサーデータ・時系列予測で優位 |

**主な用途**

- 予兆検知・劣化保全
- 異常検知
- 需要予測
