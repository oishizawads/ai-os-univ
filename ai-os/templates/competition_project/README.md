# Competition Project Template

このディレクトリは新しいコンペを始める時の雛形。

## 使い方
1. このフォルダを `competitions/<your-competition-name>/` にコピーする
2. `COMPETITION.md`, `DATASET.md`, `METRIC.md`, `SESSION_NOTES.md` を埋める
3. `.steering/` を切って baseline 実験から始める

## 重要ファイル
- `CLAUDE.md`
- `COMPETITION.md`
- `DATASET.md`
- `METRIC.md`
- `SESSION_NOTES.md`
- `VALIDATION_RULES.md`
