# University - CLAUDE.md

## Purpose
授業・課題・レポート管理。

## ディレクトリ構成
課題ごとにフォルダを切る：
```
university/
  <科目名>/
    notes.md        ← 授業ノート
    assignments/    ← 課題・提出物
    refs/           ← 参考資料メモ
```

## 課題を始めるときに確認すること
- [ ] 締切はいつか
- [ ] 成果物の形式（字数・形式・提出先）
- [ ] 評価基準
- [ ] 参照可能な資料の範囲

## AIの使い分け
- 概念理解 → Claude に説明させる（自分の言葉で言い直して確認）
- 調査 → `researcher` で先行研究を拾う
- 構成 → アウトラインだけ Claude、本文は自分
- レビュー → 書いたものを Claude に論理チェックさせる
