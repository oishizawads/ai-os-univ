# Grad Exam - CLAUDE.md

## Purpose
院試対策。科目別の理解・過去問・進捗管理。

## ディレクトリ構成
```
grad_exam/
  <科目名>/
    notes.md        ← 理解まとめ
    problems/       ← 過去問・練習問題
    weak_points.md  ← 苦手・未理解リスト
  progress.md       ← 全科目の進捗トラッカー
```

## 勉強セッションの進め方
1. `weak_points.md` を見て今日の対象を決める
2. Claude に概念を説明させて理解を確認
3. Claude に問題を作らせて自分で解く
4. 解説を見て `notes.md` を更新
5. まだ曖昧なら `weak_points.md` に残す

## AIの使い方
- 概念説明 → 「〇〇を5行で説明して。その後確認問題を1問出して」
- 過去問解説 → 解いてから Claude に解説させる（先に答えを見ない）
- 弱点特定 → 「〇〇の中で特につまずきやすいポイントを列挙して」
- 模擬問題 → 「〇〇の典型問題を3問作って。難易度は院試レベル」

## 進捗トラッカーの書き方
```markdown
| 科目 | 理解度 | 過去問 | 最終確認日 |
|------|--------|--------|-----------|
| 〇〇 | ★★★☆☆ | 3年分 | 2026-04-05 |
```
