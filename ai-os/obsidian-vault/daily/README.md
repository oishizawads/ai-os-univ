# Daily Notes

日常で触れた情報・気づき・メモの置き場。
Obsidian の Daily Notes と連携させて使う。

## 対象
- 読んだ記事の感想・要点
- 会話・議論で気づいたこと
- ふと思った疑問
- 試してみたいこと

## ファイル命名
`YYYY-MM-DD.md`

## 最小テンプレート
```markdown
# YYYY-MM-DD

## 気づき
-

## 読んだもの
-

## 疑問・調べたいこと
-

## アイデア → ideas/ に移す候補
-
```

## 週次レビューでやること
- `ideas/` に昇格させるものを移す
- `obsidian-vault/raw/` にクリップすべき記事を ingest する
- 繰り返し出てくるテーマを `knowledge/` 化する
