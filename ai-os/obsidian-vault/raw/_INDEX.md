# Obsidian Raw — Index

Web Clipperでクリップした記事・論文・メモの目次。
新しいファイルを追加したらここに1行追記する。

---

## 使い方

| 操作 | 方法 |
|------|------|
| 記事を追加 | Obsidian Web Clipper → `raw/` に保存 → この index に追記 |
| 検索 | `knowledge-pipeline --query "..."` |
| wiki化 | `knowledge-pipeline --compile` |

## ディレクトリ構成

| フォルダ | 内容 |
|---------|------|
| `inbox/` | 未分類のクリップ |
| `blogs/` | ブログ記事 |
| `papers/` | 論文・技術レポート |
| `notebooks/` | Jupyter / 分析メモ |
