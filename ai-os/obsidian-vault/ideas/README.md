# Ideas

思いついたアイデア・気づき・仮説を書き捨てる場所。
完成度は問わない。後で `knowledge/` に昇格させるかどうかを判断する。

## 書き方（最小形式）
```markdown
## アイデアタイトル
日付: YYYY-MM-DD

何を思いついたか（1〜3行）

なぜ面白いと思ったか

次にやるとしたら何をするか
```

## 昇格ルール
- 汎用的に使えそう → `ai-os/knowledge/principles/` or `frameworks/`
- 案件・コンペに使えそう → 該当プロジェクトの `SESSION_NOTES.md`
- ビジネスアイデア → `ai-os/work/zetax/proposals/`
- 研究アイデア → `ai-os/study/research/`
- そのまま放置でもいい（アイデアは量が大事）
